# Pipeline 执行指南

## OpenClaw Pipeline 配置

本 Pipeline 通过 OpenClaw 的 HEARTBEAT 机制执行，无需额外脚本。

## 执行时间

| 执行点 | 时间 | 说明 |
|-------|------|------|
| 午盘监控 | 11:30 | 上午收盘后，分析早盘情况 |
| 收盘监控 | 14:30 | 下午收盘前，分析全天情况 |

## 任务流程

### 每次执行时：

1. **读取持仓配置**
   - 读取 `holdings.json` 获取当前持仓
   - 读取 `thresholds.json` 获取触发阈值

2. **数据采集**
   - 访问财联社获取事件日历
   - 访问东方财富获取板块动态
   - 访问新浪财经获取政策新闻

3. **信息筛选**
   - 匹配持仓相关的行业和个股新闻
   - 过滤无关信息

4. **重要性评估**
   - 按 `thresholds.json` 标准分级
   - 🔴重要 / 🟡提醒 / 🟢资讯

5. **通知决策**
   - 只有 🔴重要 和 🟡提醒 级别才生成报告
   - 🟢资讯 不生成报告（减少信息噪音）
   - 如果全天无任何重要/提醒级别信息，发送 "HEARTBEAT_OK"

6. **发送通知**
   - 🔴重要：飞书消息 + 邮件 (squellwww@me.com)
   - 🟡提醒：仅飞书消息

## 报告格式

### 飞书消息格式

```
📊 投资监控简报 | 2026-03-03 11:30

【重要】
• [标的] [事件] [影响]
  [操作建议]

【提醒】
• [标的] [事件]

【今日无重要变化】
• 黄金ETF、半导体ETF、新能源ETF
```

### 邮件格式（仅重要事项）

```
主题: [重要] 投资监控提醒 - 2026-03-03 11:30

详细分析...

持仓影响评估...

操作建议...

---
投资监控系统 | 自动生成
```

## HEARTBEAT.md 配置

```markdown
## 投资监控任务

### 执行时间
- 11:30 (午盘)
- 14:30 (收盘前)

### 执行步骤
1. 访问财联社 https://www.cls.cn/ 获取事件
2. 访问东方财富 https://www.eastmoney.com/ 获取行情
3. 分析持仓影响
4. 按 thresholds.json 分级
5. 如为 🔴重要或🟡提醒，生成报告并发送
6. 如无任何变化，回复 HEARTBEAT_OK

### 持仓清单
- 黄金：华夏黄金ETF联接C、黄金ETF前海开源
- AI：华夏人工智能ETF联接C
- 游戏：华夏中证动漫游戏ETF联接C
- 芯片：国泰CES半导体芯片行业ETF联接A
- 新能源：汇添富中证新能源汽车产业指数
- 军工：博时军工主题股票A
- 个股：南天信息
```

## 优化迭代

- 调整阈值敏感度
- 增减信息源
- 优化报告格式
