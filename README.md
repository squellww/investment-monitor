## 重要性分级标准

### 🔴 重要（发送飞书+邮件）
触发条件：
- 持仓标的单日涨跌超过 ±5%
- 相关行业重大政策发布（监管/补贴/禁令）
- 持仓个股公司重大公告（业绩/重组/违规）
- 宏观重大事件（美联储加息/央行降息/战争冲突）
- 建议调整定投计划或止损止盈

### 🟡 提醒（仅飞书）
触发条件：
- 相关行业新闻可能影响中期走势
- 定投扣款日前提醒
- 行业展会/发布会等重要事件预告
- 持仓连续3日同向波动超过3%

### 🟢 资讯（不通知，仅记录）
触发条件：
- 一般性行业新闻
- 市场整体走势描述
- 无明显影响的政策解读

```

【重要】
• 动漫游戏ETF今日跌X%，触发关注线
• 建议：暂停本周定投，观察后势

【提醒】  
• 明天3月4日苹果发布会，关注AI板块

【无重要变化】
• 黄金/芯片/新能源/军工 - 暂无异常
```

### 邮件模板（重要事项）
```
主题: [重要] 投资监控提醒 - 2026-03-03

详细分析...
操作建议...
```

## 配置文件

### holdings.json
用户持仓配置，支持动态更新

### sources.json  
信息源配置，可增减监控来源

### thresholds.json
触发阈值配置，可调整敏感度

## 部署说明

### 本地运行
```bash
python monitor.py --config config.yaml
```

### 定时任务（Windows Task Scheduler）
- 触发器：工作日 11:25, 14:25
- 操作：执行 run.bat

### Docker 部署
```bash
docker build -t investment-monitor .
docker run -d --env-file .env investment-monitor
```
